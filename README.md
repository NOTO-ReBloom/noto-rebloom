# NOTO Re:Bloom website

NOTO Re:BloomのGitHub Pages用静的サイトです。リポジトリ直下へ全ファイルをアップロードしてください。

- `index.html`：トップページ
- `learn.html`：耕作放棄地を知る
- `event.html`：泥ん子運動会
- `partner.html`：法人・団体向け
- `diagnosis.html`：花タイプ診断
- `site.css`：全ページ共通デザイン
- `site.js`：全ページ共通動作
- `diagnosis.js`：花タイプ診断

公開前に `index.html`、`site.css`、`site.js`、主要画像が同じ階層にあることを確認してください。

今回の版では、明朝体を廃止し、日本語ゴシック系の書体、文節を意識した改行、画面幅別の文字サイズを全ページへ適用しています。
