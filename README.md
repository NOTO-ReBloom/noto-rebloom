# NOTO Re:Bloom Website｜内容深化・企業提示版

耕作放棄地の問題を知り、泥ん子運動会への参加、READYFORでの応援、企業・団体との連携へつなげる公式サイトです。

## 主なページ
- `index.html`：プロジェクト全体、準備の裏側、9月20日までの準備ボード
- `learn.html`：耕作放棄地・荒廃農地の基礎、全国データ、能登の復旧動向、活用事例、4問クイズ
- `event.html`：泥ん子運動会、レンゲカップ、安全設計、参加後に持ち帰る学び
- `partner.html`：法人・団体向け協賛・連携、業種別協力例、復興の中での役割、準備の根拠
- `diagnosis.html`：56問・32タイプのRe:Bloom花タイプ診断

## 今回の内容強化
- 全国の荒廃農地について「総面積」だけでなく、年間の発生と再生、農地面積の長期変化を追加
- 能登の農業復旧について、2026年の石川県公式情報を追加
- 荒廃農地の活用方法を、集積・省力作物・放牧・蜜源作物・市民農園・地域協議に整理
- レンゲの緑肥としての歴史と、今回カップ方式を採る理由を明示
- 準備過程、安全相談、活動報告を可視化
- 花診断に5軸の文章解説、近い別タイプ、補完関係、30日プランを追加

公開手順は `DEPLOY_GUIDE.md`、確認内容は `QUALITY_CHECK.md`、根拠資料は `CONTENT_SOURCES.md` を参照してください。
