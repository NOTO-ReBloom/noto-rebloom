# GitHub Pagesへの反映手順

1. このZIPを解凍します。
2. GitHubの `noto-rebloom/noto-rebloom` リポジトリを開きます。
3. `Add file` → `Upload files` を選びます。
4. 解凍したフォルダの中身をすべて選択し、リポジトリの最上位へアップロードします。
5. `Commit changes` を押します。
6. 数分待ち、公開サイトを `Ctrl + Shift + R` で強制再読み込みします。

## 必ず上書きするファイル
- index.html
- learn.html
- event.html
- partner.html
- diagnosis.html
- site.css
- site.js
- diagnosis.js
- 404.html
